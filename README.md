# 本草 · 养身

一个基于**中医体质辨识**的中药 · 熏香养身参考程序，纯本地网页，双击即用，无需安装环境、无外部依赖。

> ⚠ 本程序为中医养生科普与体质辨识参考工具，**不构成任何医疗诊断、治疗建议或处方**。所有建议仅供日常养身参考；孕妇、慢性病患者、过敏体质及服药者使用任何本草、艾灸、熏香、穴位调理前请咨询执业中医师。

## 运行方式

### 🖥 桌面应用（推荐）
- **桌面已有快捷方式「本草·养身」**，双击即可在独立窗口中运行（带自定义本草图标）。
- 可执行文件：**`dist\本草养身.exe`**（约 18 MB，单文件、绿色免安装）。
- 桌面版基于 **pywebview**，使用系统自带的 Edge WebView2 内核，无需安装浏览器或运行环境。
- **重新打包**（修改代码后）：双击 `build_exe.bat`，新 exe 仍在 `dist\` 下；图标由 `make_icon.py` 生成的 `本草养身.ico`。
- **开发运行**（不打包）：`py desktop_app.py`（需先 `py -m pip install pywebview`）。
- **重建桌面快捷方式**：见下方 PowerShell 片段或重新运行创建命令。

### 🌐 网页方式
1. **零安装独立窗口**：双击 `启动-本草养身.bat`，用 Edge 应用模式打开（无浏览器边框）。
2. **直接打开**：双击 `index.html` 在浏览器中使用。
3. **本地服务器**：`py -m http.server 8123 -d "<本目录>"`，访问 http://localhost:8123

## 核心流程：问诊 → 断诊 → 出诊

| 环节 | 内容 |
|------|------|
| **问诊** | 中华中医药学会《中医体质分类与判定》标准，9 体质 · 66 题量表，分页作答带进度 |
| **断诊** | 标准转化分算法计分，判定主体质与兼夹体质，配九维雷达图与得分条 |
| **出诊** | 按体质给出四类养身建议：① 中药茶饮/代茶饮 ② 熏香/芳香理疗 ③ 食疗/药膳 ④ 起居/经络/导引 |

外加两个工具模块：
- **经络穴位图（解剖参考版）**：人体正/背面 SVG **半解剖图**——绘出肌肉分群（胸大肌、腹直肌腱划、三角肌、股四头肌、腓肠肌等），并可一键叠加**骨骼参考层**（颅骨、锁骨、胸廓肋骨、骨盆、四肢长骨；背面为完整脊柱椎骨节段并标注 C7/T3/T7/L2 编号，便于按骨性标志定位背俞穴）。两种模式：
  - **对症取穴**：14 类困扰（头痛、失眠、颈肩、感冒、脾胃、畏寒、情绪、痛经、祛湿、眼疲劳、便秘、腰酸、上火、补气等），选症状即在图上按解剖位置标出穴位，给出定位、简易取穴、主治、揉捏手法与宜忌；点击单穴看详解。
  - **全身总览·穴位大全**：海报式排版，全身一张图、42 个常用穴位用引线连到两侧标注框（穴名 + 主治简注），点击穴名跳转详解。
  共收录 42 个常用安全穴位。出诊方案"经络"板块可一键跳转到对应体质的取穴方案。
- **本草·香材功效库**：32 味常用本草与香材，含性味归经、功效、熏香/茶饮用法、宜忌，支持分类筛选与关键词搜索。

- **二十四节气养生**：按公历日期**自动判断当令节气**，首页显示"当令养生"卡片（含距下一节气天数）；节气页按四季列出 24 节气可点选，每节气含物候、养生要点、宜食、代茶饮、起居、宜忌。

出诊方案支持 **🖨 打印 / 导出**（打印时自动隐去导航与按钮，仅输出当前体质的调养方案）。

### 转化分判定标准
- 转化分 = [(原始分 − 条目数) / (条目数 × 4)] × 100
- 偏颇体质：≥40「是」、30~39「倾向是」、<30「否」
- 平和质：≥60 且其余体质均较低方判「是/基本是」

## 九种体质
平和质 · 气虚质 · 阳虚质 · 阴虚质 · 痰湿质 · 湿热质 · 血瘀质 · 气郁质 · 特禀质

```
└─ dist/本草养身.exe      打包好的桌面应用
```
